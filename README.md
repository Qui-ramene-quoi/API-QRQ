## Requirements ##

Before starting, you need to have [Node](https://nodejs.org/en/) installed.

## Starting ##

```bash
# Clone this project
$ git clone https://github.com/Qui-ramene-quoi/API-QRQ.git

# Access
$ cd api-qrq

# Install dependencies
$ npm i

# Run the project
$ npm start dev

# The server will initialize in the <http://localhost:3000>
```

## Using Docker ##

### Build image first ##

```bash
# At root folder
$ docker build --tag qrq-image/base:latest .
```

### Create network ##

```bash
# At root folder
$ docker network create main_bridge --driver bridge
```

### Start container with docker-compose

```bash
# At root folder
$ docker-compose up -d
```
